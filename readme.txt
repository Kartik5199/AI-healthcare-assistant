# AI-Based Healthcare Assistant

An AI-powered healthcare assistant for disease prediction, brain tumor detection, and medicine lookup.

## Features
✅ Disease prediction based on symptoms  
✅ Brain tumor detection via MRI upload  
✅ Chatbot for user queries (Powered by LLM API)  
✅ Medicine lookup by name or disease  
✅ User authentication & dashboard  
✅ Dark mode & green-white theme  

## Tech Stack
- **Frontend:** HTML, Tailwind CSS  
- **Backend:** Flask, MongoDB  
- **AI Models:** Scikit-learn (Disease Prediction), TensorFlow (Brain Tumor Detection)  
- **Chatbot:** LLM API
